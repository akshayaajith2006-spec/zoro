# zoro
Wi-Fi event Booking System
## Wi-Fi Event Booking System

A web-based college event booking system built for a campus environment.

## Team Roles

### Member 1 – Frontend
Files:
- index.html
- admin.html
- dashboard_user.html
- css/styles.css

Responsibilities:
- UI design
- Page layout
- Styling
- Frontend structure only (no backend logic)

### Member 2 – Backend / Firebase
Files:
- js/firebase-config.js
- js/auth.js
- js/admin.js
- js/booking.js

Responsibilities:
- Firebase setup
- Authentication
- Firestore database
- Real-time seat booking log
